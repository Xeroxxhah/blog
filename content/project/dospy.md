A Simple yet powerfull Dos script.

## Installation:
- Just git clone it using ```git clone https://github.com/Xeroxxhah/DOS.PY.git```
- Dos.py has only one dependency,i.e scapy.you can install it using ```python3 -m pip install scapy```.
